Порядок заливки базы мира:

Ядро оптимизировано под базу YTDB.
Обычно, при мердже с офф репо я пишу, под какую версию YTDB я подгоняю ядро.
Чтобы собрать базу нужно взять эту версию, залить на нее файлы из папки sql/updates/world, если их много, их можно все объединить в один файл с помощью батника.
Потом  нужно залить файлы из sql/updates/Arcanum в таком порядке:

[INDEV] Gunship_Battle.sql
[INDEV] Piligrims_Bounty.sql
world_FULL_число.sql
update_400+.sql

Порядок заливки базы персонажей:

Залить фул sql/base/characters_database.sql
Залить ап sql/Arcanum/characters_FULL.sql

Порядок заливки базы auth:
Залить фул sql/base/auth_database.sql
